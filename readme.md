# Découverte CSS

Ce site sert de base de départ pour un atelier visant à faire découvrir le CSS en remaniant son fichier `style.css`.

Un aperçu du site non remanié est disponible sur https://labasse.github.io/decouverte-css .

Au fur et à mesure des manipulations de l'atelier, le site gagnera en modernité sans une seule modification sur son code HTML.
